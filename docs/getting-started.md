# Getting Started

## Installation

```bash
git clone https://github.com/clanker-grid/clanker-grid.git
cd clanker-grid
npm install
```

## First Launch

1. Run `npm run dev`
2. The workspace gate opens if no workspaces exist
3. Select or enter a directory path
4. Choose terminal count (1, 2, or 4)
5. Optionally select an AI harness
6. Click **Launch**

## Creating Workspaces

From the gate or header toolbar:
- Click **Open Workspace**
- Enter a local directory path
- Directory autocomplete is available

## Navigation

| Element | Location |
|---------|----------|
| Title Bar | Top — window controls |
| Workspace Tabs | Below title bar |
| Header Toolbar | Tabs → main content |
| Main Area | Terminals and browser |
| Status Bar | Bottom |

## Quick Commands

| Command | Action |
|---------|--------|
| `npm run dev` | Start development |
| `npm run build` | Build for production |
| `npm run test` | Run tests |
| `npm run validate` | Full validation |
