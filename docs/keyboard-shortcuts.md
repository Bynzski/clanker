# Keyboard Shortcuts

## Global

| Action | Shortcut |
|--------|----------|
| New Terminal | `Ctrl+Shift+T` |
| Close Workspace | `Ctrl+W` |
| Open Workspace | `Ctrl+O` |
| Toggle Browser | `Ctrl+B` |
| Settings | `Ctrl+,` |

## Terminal

| Action | Shortcut |
|--------|----------|
| Copy | `Ctrl+Shift+C` |
| Paste | `Ctrl+Shift+V` |
| Clear | `Ctrl+L` |

## Pane Layout

| Action | Description |
|--------|-------------|
| Drag handle | Move pane |
| Double-click divider | Fit all panes |

## Git

| Action | Shortcut |
|--------|----------|
| Open Git Menu | `Ctrl+G` |
| Commit | `Ctrl+Enter` (in dialog) |

## Browser

| Action | Shortcut |
|--------|----------|
| Back | `Alt+←` |
| Forward | `Alt+→` |
| Refresh | `Ctrl+R` |
| Open External | `Ctrl+Shift+O` |
| DevTools | `Ctrl+Shift+I` |
| DevTools (alt) | `F12` |
| Fit Panes | `Ctrl+Shift+F` |

## Editor

| Action | Shortcut |
|--------|----------|
| Save File | `Ctrl+S` |
| Close Tab | `Ctrl+W` (when editor focused) |
